# Les Éclaireurs ⚡

Plateforme nationale de formation pour la filière électrique française.

## Stack
- React 18 + Vite
- CSS-in-JS (styles embarqués)
- Aucune dépendance UI externe

## Lancer en local

```bash
npm install
npm run dev
```
→ Ouvre http://localhost:5173

## Déployer sur Vercel

1. Pusher ce repo sur GitHub
2. Connecter sur vercel.com
3. Déploiement automatique ✓
